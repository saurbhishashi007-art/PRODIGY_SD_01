# Temperature Converter

## Overview

This is a simple Java-based Temperature Converter that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

* Celsius to Fahrenheit
* Fahrenheit to Celsius
* Celsius to Kelvin
* Kelvin to Celsius
* User-friendly menu-driven interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.
2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, VS Code, etc.).
3. Compile and run the Java file.
4. Enter the temperature value and select the conversion type.

## Sample Conversions

* 25°C = 77°F
* 100°F = 37.78°C
* 0°C = 273.15 K
